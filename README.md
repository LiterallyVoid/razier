
# Razier

Version 0.1.0

[Github](https://literallyvoid.github.io/razier)

_Arcade fast paced hover racing game._

By **LiterallyVoid** and **KaadmY**

## Help

Controls

- **W** key to accelerate

- **S** key to brake

- **A** and **D** keys to turn
